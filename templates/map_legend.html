Citi Bike trips visualized.

This project visualizes Citi Bike data from March 2024. It was built with Python, using plotly.py for statistical analysis and the folium for maps. Visualize the data using the options below.

  • map by bike type
  • map by time of day
  • 2d histogram of stations
  • scatter plot of stations
